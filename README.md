# DinoGame-project
Endless runner game inspired by the classic Chrome dinosaur game.
Developed in the Godot Engine as a personal project.

## Features

- Character Jumping
- Obstacle Generation
- Scoring System
- Game Over Screen

## Technologies

- Godot Engine 4.5
- GDScript
- Scene and Node System
- Character Collision Detection

## How to Run the Project

### Option 1 – Download Executable
1. Go to the **Releases** section
2. Download the .exe file
3. Run the game

### Option 2 – Open in Godot
1. Clone the repository
2. Open the project from within Godot
3. Run the main scene

## Controls

- SPACE to jump
- SPACE on the Game Over screen to restart
